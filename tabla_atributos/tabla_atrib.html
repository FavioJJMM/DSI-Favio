Tabla de atributos
Nombre del Atributo Funcionalidad
class Este atributo se utiliza para definir una clase CSS para un elemento. Las clases se utilizan para aplicar estilos específicos a un conjunto de elementos.
id Este atributo se utiliza para definir un identificador único para un elemento. Los identificadores se utilizan para seleccionar un elemento específico y aplicar estilos o scripts específicos a ese elemento.
style Este atributo se utiliza para aplicar estilos en línea a un elemento. Puede utilizar este atributo para aplicar estilos específicos a un elemento sin tener que definir una clase CSS o utilizar un archivo de estilo externo.
title Este atributo se utiliza para proporcionar información adicional sobre un elemento. Se muestra un mensaje emergente con el texto del atributo "title" cuando se coloca el cursor sobre el elemento. Este atributo se utiliza a menudo para proporcionar información adicional sobre enlaces o imágenes.
src Este atributo se utiliza para especificar la ruta de la fuente (imagen, audio, video, etc.) que se va a mostrar en el elemento. Por ejemplo, puede utilizar este atributo para especificar la ubicación de una imagen que se mostrará en una página web.
href Este atributo se utiliza para especificar la dirección URL a la que se va a dirigir el enlace. Puede utilizar este atributo para crear enlaces a otras páginas web o a otros recursos en línea.
alt Este atributo se utiliza para proporcionar un texto alternativo que se muestra si la imagen no se puede cargar. Este texto alternativo se utiliza a menudo para describir el contenido de una imagen para los usuarios de tecnologías de asistencia.
data-* Este atributo se utiliza para almacenar datos privados de la página o aplicación en el elemento. Puede utilizar este atributo para guardar información adicional sobre un elemento que se puede utilizar más tarde en scripts o para interactuar con otros componentes de la página.
target Este atributo se utiliza para especificar dónde se abrirá el enlace cuando se haga clic en él. Puede utilizar este atributo para abrir un enlace en una nueva ventana del navegador, en la misma ventana o en un marco específico de la página.
type Este atributo se utiliza para especificar el tipo de contenido que se encuentra en el elemento. Puede utilizar este atributo para especificar el tipo de datos en un elemento de entrada, como un campo de entrada de texto o un botón de envío, o para especificar el tipo de contenido en un elemento de estilo o de script.
value Este atributo se utiliza para especificar el valor predeterminado o el valor actual de un elemento de entrada. Puede utilizar este atributo para establecer el valor predeterminado en un campo de entrada o para obtener el valor actual de un elemento de entrada en un script.
