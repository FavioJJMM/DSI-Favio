Etiquetas y atributos HTML

A continuación se muestran algunas etiquetas HTML comunes y algunos de sus atributos:

Etiquetas multimedia:

<video>: Se utiliza para insertar videos en una página web. Algunos de sus atributos son:

  • src: especifica la URL del video
  • poster: especifica una imagen de vista previa para el video
  • width: especifica el ancho del video en píxeles
  • height: especifica la altura del video en píxeles
  • autoplay: hace que el video se reproduzca automáticamente
  • controls: muestra los controles de reproducción del video
  • loop: hace que el video se reproduzca en bucle
  • preload: indica si el video debe ser cargado al cargar la página (none, metadata, auto)

<iframe>: Se utiliza para insertar contenido de otras páginas en una página web. Algunos de sus atributos son:

  • src: especifica la URL de la página a ser mostrada en el iframe
  • width: especifica el ancho del iframe en píxeles
  • height: especifica la altura del iframe en píxeles
  • frameborder: indica si el borde del iframe debe ser mostrado o no (0 o 1)
  • allowfullscreen: permite que el iframe se pueda ver en pantalla completa (solo para videos)

Etiquetas para estructurar contenido:

<ol>: Crea una lista ordenada. Algunos de sus atributos son:

  • start: especifica el número de inicio de la lista ordenada
  • type: especifica el tipo de marcador de la lista (1, A, a, I, i)
  • reversed: invierte el orden de la lista ordenada

Ejemplo:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

<li>: Define un elemento de la lista. No tiene atributos propios, pero puede heredar algunos de la etiqueta `ol` o `ul`.

Ejemplo:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Etiquetas para enlace:

<a>: Se utiliza para crear enlaces a otras páginas web o a diferentes secciones de la misma página. Algunos de sus atributos son:

  • href: especifica la URL del destino del enlace
  • target: indica dónde se abrirá el enlace (en la misma ventana o en una nueva ventana/tab) (_self o _blank)
  • title: añade un texto que aparece cuando se pasa el cursor sobre el enlace

Ejemplo:

Ir a Google

Etiquetas para tablas:

<table>: Se utiliza para crear tablas en una página web. Algunos de sus atributos son:

  • border: especifica el grosor del borde de la tabla en píxeles
  • cellspacing: especifica el espacio entre las celdas de la tabla en píxeles
  • cellpadding: especifica el espacio entre el contenido de las celdas y el borde de las mismas en píxeles
  • width: especifica el ancho de la tabla en píxeles o en porcentaje
  • height: especifica la altura de la tabla en píxeles o en porcentaje

Ejemplo:

Nombre Edad
Juan 25
María 30

<col>: Define las propiedades de una o varias columnas de una tabla. Algunos de sus atributos son:

  • span: especifica cuántas columnas debe abarcar el <col>
  • width: especifica el ancho de la columna
  • style: modifica el estilo de la columna
  • align: especifica la alineación del contenido de la columna (left, center, right, justify, char)

Ejemplo:

Nombre Edad País
Juan 25 México
María 30 Colombia

Etiquetas para formularios:

<form>: Se utiliza para crear formularios en una página web. Algunos de sus atributos son:

  • action: especifica la URL a la que se enviarán los datos del formulario
  • method: especifica el método HTTP que se utilizará para enviar los datos del formulario (get o post)
  • enctype: especifica cómo se codificarán los datos del formulario antes de enviarlos (application/x-www-form-urlencoded o multipart/form-data)
  • name: especifica el nombre del formulario

<input>: Se utiliza para crear un campo de entrada en un formulario. Algunos de sus atributos son:

  • type: especifica el tipo de campo de entrada (texto, email, contraseña, etc.)
  • id: especifica un identificador único para el campo de entrada
  • name: especifica el nombre del campo de entrada
  • placeholder: especifica una breve descripción del campo de entrada
  • required: especifica que el campo de entrada debe ser rellenado antes de enviar el formulario
